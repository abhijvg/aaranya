# Logo Image

Please place your logo image file here with the name `logo.png`.

The logo should be:
- Format: PNG (with transparency) or JPG
- Recommended size: 360x120px or larger (will be scaled down)
- File name: `logo.png`

The logo will be used in:
- Main site header
- Admin dashboard header

If you have the logo in a different format, you can:
1. Convert it to PNG
2. Or update the file extension in `components/Header.tsx` and `components/admin/AdminNav.tsx`
